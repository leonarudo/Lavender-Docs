# Your First Data Mod
<br>

![Static Badge](https://img.shields.io/badge/Skill_Level-Beginner-blue?style=for-the-badge)  

![Static Badge](https://img.shields.io/badge/Estimated_Time-30_Minutes-blue?style=for-the-badge)

**You will learn:**

- How to create an Lavender Data mod
- How to add modded items and recipes to Obenseuer without using code
- How to add deco furniture to Obenseuer without using code

<br>

> [!NOTE]  
> **Data Mods** are an easy and beginner-friendly way to add your own items, recipes, and furniture to **Obenseuer**. They are perfect if you have little or no experience with C# or programming in general.  
>  
> Just be aware that **Data Mods are limited** — they can’t add new game logic or complex behavior.  
>  
> If you want to create more advanced or custom features, check out the [Your First Mod](./Your-First-Mod.md) guide to get started.
